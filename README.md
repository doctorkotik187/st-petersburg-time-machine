# St. Petersburg Time Machine

An interactive map of Saint Petersburg with a timeline slider. Explore the city's history through its iconic landmarks — from the Peter and Paul Fortress to the Russian Museum.

## Features

- Interactive map with landmark markers
- Timeline slider (1700–2026) — filter places by year built
- Russian-language descriptions for each landmark

## Run

```bash
npm install
npx shadow-cljs watch app
```

Open http://localhost:8080

## Stack

ClojureScript, shadow-cljs, Reagent, Leaflet
